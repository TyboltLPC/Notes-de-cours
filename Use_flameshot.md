# Flameshot outil de capture d'écran
___
___
## install
`apt install flameshot`

## Config

`flameshot config -a true`   start auto et accés config

* choix couleur
* choix du nom (%F-%Hh%M)
* choix outils

## Racourcis

* Dans clavier > racourcis
* Nouveau personnaliser
* Commande `flameshot gui -p /chemin/de/stockage`
* Attibuer une touche
